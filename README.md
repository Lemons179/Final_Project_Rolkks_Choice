# Final_Project_Rolkks_Choice
Final Project for Game Programming Spring 2026<br/>

This game is a platformer wherein the player must collect 7 coins in order to unlock an exit door that will allow them to escape the forest they are in and win the game.
The player must avoid enemies by jumping around them or alternatively may kill enemies by firing projectiles at them. If the player is attacked by enemies and sustains too much damage such that their health reaches zero, the player will receive a game over.<br/>
